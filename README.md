# 🌡️ SNO_HeatFlow - Intelligente Heizungssteuerung

[![Open your Home Assistant instance and show the blueprint import dialog with a specific blueprint pre-filled.](https://my.home-assistant.io/badges/blueprint_import.svg)](https://my.home-assistant.io/redirect/blueprint_import/?blueprint_url=PLATZHALTER_FUER_DEINE_RAW_GITHUB_URL)

**SNO_HeatFlow** ist ein hochmoderner, extrem flexibler Blueprint für Home Assistant, der die Heizungssteuerung auf das nächste Level hebt. Er ermöglicht die Einrichtung detaillierter und fehlertoleranter Zeitpläne pro Raum oder Zone, ohne unübersichtliche Automatisierungen schreiben zu müssen.

## ✨ Hauptfunktionen
* **Zonenbasierte Steuerung:** Wähle einzelne Thermostate, komplette Geräte oder ganze Räume.
* **Flexible Zeitfenster:** Definiere bis zu 7 Zeitfenster (Schichten) mit individuellen Temperaturen und weise diesen beliebig viele Wochentage zu.
* **Smarter Abwesenheits-Modus:** Pausiere den Zeitplan dynamisch über einen Schalter. Temperaturen können wahlweise über Dashboard-Helfer oder direkt im Blueprint festgelegt werden.
* **Mächtiger Urlaubs-Modus:** Vollständige Kalender-Integration oder Steuerung per Schalter. Inklusive sekundengenauer Start- und Endzeitpunkte (wahlweise über Helfer-Entitäten oder fixe UI-Eingaben).
* **Intelligentes Rollback:** Wenn heute noch kein Zeitfenster erreicht wurde, blickt die Logik automatisch in die vergangenen Tage zurück, um die korrekte Nachttemperatur zu ermitteln.
